# bookKeeping

免费开源的智能记账 Android APP，主打自动记账能力。

## 功能

- 📝 **手动记账** - 快速录入收支记录
- 📸 **截图 OCR 识别** - 拍摄/选择账单截图自动识别金额、商户
- 🔔 **通知栏自动记账** - 监听支付宝、微信等支付通知
- 📱 **短信自动记账** - 监听银行扣款短信
- ♿ **无障碍自动记账** - 从支付APP读取账单（需逐APP配置）
- 📊 **统计报表** - 按日/周/月/年查看收支统计和分类占比
- 💾 **数据导入导出** - JSON 格式导入导出，支持支付宝/微信 CSV 账单导入
- 🎯 **智能分类** - 根据商户名自动匹配消费分类

## 截图识别触发方式

- 悬浮球按钮
- 截图自动识别
- 敲击背面触发（需系统 Quick Tap 设置）

## 技术栈

- Kotlin + Jetpack Compose
- MVVM + Room + Coroutines
- ML Kit OCR（中文识别）
- Hilt 依赖注入
- Material 3 设计

## 构建

```bash
./gradlew assembleDebug
```

## 最低要求

- Android 8.0 (API 26) 以上
- 部分自动记账功能需要 Root 权限
